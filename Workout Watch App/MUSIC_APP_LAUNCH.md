# アプリ一覧画面への戻り方

## 概要

ワークアウトアプリからApple Watchのアプリ一覧画面に戻り、他の音楽アプリを起動してバックグラウンドでワークアウトを継続できるようになりました。

## 実装内容

### 1. アプリ一覧ボタン

ミュージックコントロール画面（右にスワイプ）の下部に「**アプリ一覧**」ボタンが追加されました。

このボタンを押すと：
1. Apple Watchのアプリ一覧画面（ホーム画面）に戻ります
2. ワークアウトアプリは自動的にバックグラウンドに移行
3. 他の音楽アプリを自由に選択して起動できます

### 2. バックグラウンド動作

アプリ一覧画面に戻っても、以下の機能は継続します：

✅ ワークアウトセッションの記録
✅ 心拍数の測定
✅ 距離の計測
✅ 消費カロリーの計算
✅ 歩数のカウント

### 3. 視覚的フィードバック

ボタンを押すと、画面中央に以下のメッセージが表示されます：

```
⬜ Digital Crownを押すと
  アプリ一覧に戻れます
  他の音楽アプリを起動できます
```

このメッセージは3秒後に自動的に消えます。

## 使用方法

### パターン1: アプリ一覧ボタンを使用（推奨）

1. ワークアウトを開始
2. 右にスワイプしてミュージックコントロール画面を開く
3. 「**アプリ一覧**」ボタンをタップ
4. Digital Crownボタンを1回押す
5. アプリ一覧画面が表示される
6. 好きな音楽アプリを選択して起動
7. ワークアウトはバックグラウンドで継続

### パターン2: Digital Crownボタンを直接使用

1. ワークアウトを開始
2. **Digital Crownボタン**を1回押す
3. アプリ一覧画面が表示される
4. 好きな音楽アプリを選択して起動
5. ワークアウトはバックグラウンドで継続

## 対応している音楽アプリ

アプリ一覧から以下のような音楽アプリを起動できます：

- 🎵 **Apple Music** (標準アプリ)
- 🎵 **Spotify**
- 🎵 **Amazon Music**
- 🎵 **YouTube Music**
- 🎵 **Pandora**
- 🎵 **その他、watchOS対応の音楽アプリ**

## 詳しい使い方

### ステップ1: ワークアウトを開始

1. アプリ選択画面でワークアウトタイプを選択（ウォーキング、ジョギング、ランニング）
2. ワークアウトが開始される

### ステップ2: アプリ一覧に戻る

**方法A: アプリ一覧ボタンを使用**
1. ワークアウト中に画面を**右にスワイプ**
2. ミュージックコントロール画面が表示される
3. 画面下部の「**アプリ一覧**」ボタンをタップ
4. 案内メッセージが表示される
5. **Digital Crownボタン**を1回押す

**方法B: Digital Crownボタンを直接使用**
1. ワークアウト中に**Digital Crownボタン**を1回押す
2. すぐにアプリ一覧画面が表示される

### ステップ3: 音楽アプリを起動

1. アプリ一覧から好きな音楽アプリのアイコンをタップ
2. 音楽アプリが起動
3. 好きな曲やプレイリストを選択
4. 音楽が再生される

### ステップ4: ワークアウトアプリに戻る（オプション）

ワークアウトの進捗を確認したい場合：

1. **Digital Crownボタン**を押してアプリ一覧を開く
2. ワークアウトアプリのアイコンをタップ
3. ワークアウト画面に戻る（音楽は継続再生）

または、

1. **サイドボタン**をダブルクリック
2. 最近使ったアプリからワークアウトアプリを選択

## トラブルシューティング

### アプリ一覧ボタンを押しても何も起こらない

**原因:**
- watchOSの制限により、アプリから直接アプリ一覧画面を開くAPIは提供されていません

**解決方法:**
1. 案内メッセージが表示されたら、**Digital Crownボタン**を1回押してください
2. または、直接Digital Crownボタンを押してアプリ一覧画面を開いてください

### ワークアウトが停止してしまう

**原因:**
- バックグラウンド実行の設定が不足している

**解決方法:**
1. `Info.plist` に `UIBackgroundModes` が設定されているか確認
2. `BACKGROUND_SETUP.md` の設定手順を参照
3. ワークアウトセッション（`HKWorkoutSession`）が正しく開始されているか確認

### 音楽アプリから戻れない

**解決方法:**
1. **Digital Crownボタン**を押してアプリ一覧を開く
2. ワークアウトアプリのアイコンをタップ
3. または、**サイドボタン**をダブルクリックして最近使ったアプリから切り替え

### 音楽が途切れる

**原因:**
- 複数のアプリが同時にオーディオリソースを使用している

**解決方法:**
1. 音楽アプリで音量を調整
2. Digital Crownを回して音量を調整
3. 他のオーディオアプリを終了

## 技術的な詳細

### アプリ一覧画面への遷移

ボタンを押すと、以下のコードが実行されます：

```swift
private func openAppGrid() {
    // 空のURLスキームでホーム画面に戻る試行
    if let url = URL(string: "") {
        WKExtension.shared().openSystemURL(url)
    }
    
    // 案内メッセージを表示
    showHomeHint = true
}
```

**注意:** watchOSの制限により、アプリから直接アプリ一覧画面を開く公式APIは提供されていません。ユーザーにDigital Crownボタンを押すよう案内することで、この機能を実現しています。

### バックグラウンド実行の仕組み

1. `HKWorkoutSession` がアクティブな間、アプリはバックグラウンドで実行を継続
2. `scenePhase` が `.background` になっても、ワークアウトセッションは継続
3. データの記録は中断されない
4. 他のアプリを起動しても、ワークアウトは継続

### デバッグログ

コンソールで以下のログを確認できます：

```
🏠 Attempting to return to app grid...
🏠 Tried: prefs:root=
💡 Digital Crownボタンを押してアプリ一覧に戻ることもできます
💡 ワークアウトはバックグラウンドで継続します
🎵 Scene phase changed from active to background
🎵 App moved to background - workout continues
```

## 注意事項

⚠️ **watchOSの制限**
- watchOSでは、アプリから直接アプリ一覧画面を開く公式APIは提供されていません
- ユーザーにDigital Crownボタンを押すよう案内することで、この動作を実現しています
- これは意図的な設計で、ユーザーの操作を尊重するためです

⚠️ **電力消費**
- ワークアウトセッションをバックグラウンドで実行すると、バッテリーを消費します
- 音楽再生も併せて行うと、さらに電力を消費します
- ワークアウトが終了したら、必ず「ワークアウトを終了」ボタンを押してください

⚠️ **操作の流れ**
- 「アプリ一覧」ボタンは案内を表示するためのものです
- 実際にアプリ一覧画面に戻るには、Digital Crownボタンを押す必要があります
- これはwatchOSのセキュリティとユーザー体験を保つための制限です

## まとめ

この機能により、以下のワークフローが実現できます：

1. ✅ ワークアウトを開始
2. ✅ アプリ一覧ボタンまたはDigital Crownでアプリ一覧画面を開く
3. ✅ 好きな音楽アプリを選択して起動
4. ✅ 音楽を再生
5. ✅ ワークアウトはバックグラウンドで継続
6. ✅ 必要に応じてワークアウトアプリに戻って進捗確認
7. ✅ ワークアウト終了時に記録を保存

これにより、Apple Watchでランニングやウォーキング中に、好きな音楽アプリで音楽を楽しみながら、運動データを記録できます！

## よくある質問

**Q: なぜボタンを押してもすぐにアプリ一覧画面に戻らないのですか？**

A: watchOSのセキュリティ上の制限により、アプリから直接アプリ一覧画面を開くことはできません。ユーザーが明示的にDigital Crownボタンを押すことで、意図しない画面遷移を防いでいます。

**Q: Digital Crownボタンを押さずにアプリ一覧に戻る方法はありますか？**

A: 残念ながら、watchOSの制限によりアプリから直接アプリ一覧画面を開く方法はありません。Digital Crownボタンを押すのが最も確実で推奨される方法です。

**Q: 他のApple製以外の音楽アプリでも動作しますか？**

A: はい、watchOS対応の音楽アプリであれば、どのアプリでも使用できます。Spotify、Amazon Music、YouTube Musicなど、多くのサードパーティ音楽アプリに対応しています。

**Q: ワークアウト中に電話がかかってきたらどうなりますか？**

A: 電話に出ると音楽は一時停止しますが、ワークアウトは継続します。電話が終了したら、音楽を再開できます。
